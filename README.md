NLF框架
===

NLF框架是六特尔独自开发的轻量级java框架，它只有1个jar包，含源代码在内只有600+KB的大小。它无侵入，无依赖，零配置，无注解，支持AOP，自动IOC，class动态加载，原生支持Oracle、SQLSERVER、MYSQL、MONGO和独创的CSV数据库。使用它你将会体验到前所未有的开发效率，并且你会发现，它支持javase和javaee应用，你甚至不需要启动Server就可以对web应用后台代码进行调试。
> 
六特尔并不愿意其他人来直接修改NLF框架的代码，因为六特尔致力于将它打造为完美的作品，其他人写的代码，实在没有加入进来的意义。
> 
但是您可以当小白鼠，提意见，提bug，好的idea我还是愿意接受的。
